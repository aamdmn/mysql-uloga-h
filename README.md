## Getting Started

Move `.env.example` variables into `.env`

```bash
mv .env.example .env
```

Run project:

```bash
# Install dependencies
bun i
# Start dev server
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Stack:

- MysqlDB - Planetscale
- Email API - Resend
- Framework - Next.js
